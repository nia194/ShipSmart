# Customs Documentation

ShipSmart knowledge base — advisory guidance, not legal or customs-broker advice.

## Core documents

- **Commercial invoice** — the primary customs document for goods sold. States
  buyer/seller, a clear goods description, HS code, country of origin, quantity, unit
  value, currency, and Incoterms. Required for most dutiable international shipments.
- **Proforma invoice** — used for non-sale shipments (samples, gifts, replacements);
  still must carry a realistic declared value for customs.
- **Packing list** — itemizes cartons, weights, and dimensions; helps inspection.

## Postal vs commercial forms

- **CN22** — short customs declaration for low-value postal items.
- **CN23** — fuller customs declaration for higher-value postal items, usually attached
  with a **CP71** dispatch note.
- Commercial courier shipments generally travel on a commercial invoice rather than CN
  forms.

## Description and value rules

Customs needs an **honest, specific** goods description ("men's cotton t-shirts", not
"clothing") and an accurate **declared value**. Undervaluation to reduce duty is a
violation and can trigger penalties, seizure, or audits. A declared value is mandatory —
an international shipment without a stated value cannot be cleared.

## Country of origin and HS code

The **country of origin** (where goods were made, not shipped from) and the **HS code**
together drive the duty rate and any trade-agreement preference. Misdeclared origin can
forfeit preferential rates.

## Export filing

Higher-value exports may need electronic export filing — for example, US shipments above
the **US$2,500** per-Schedule-B threshold generally require an EEI filed through AES.
Always confirm the current threshold and exemptions for the lane.

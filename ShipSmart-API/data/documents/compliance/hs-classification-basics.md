# HS Classification Basics

ShipSmart knowledge base — advisory guidance; classification is shipper responsibility.

## What an HS code is

The **Harmonized System (HS)** is a global, standardized numbering scheme for traded
goods, maintained by the World Customs Organization. The first **6 digits are
international**; countries append further digits for their own tariff and statistics
(e.g. the US Schedule B / HTS adds up to 10 digits).

## Structure

- **Chapter** (2 digits) — broad category, e.g. 85 = electrical machinery.
- **Heading** (4 digits) — a group within the chapter, e.g. 8507 = electric accumulators.
- **Subheading** (6 digits) — the specific good, e.g. 8507.60 = lithium-ion accumulators.

## Why classification drives everything

The HS code determines the **duty rate**, eligibility for **trade-agreement preference**,
and whether **import controls** (licenses, certifications) apply. A wrong code can mean
overpaying duty, underpaying (a violation), or missing a required permit.

## Representative codes

- Power bank / lithium-ion accumulator — **8507.60**
- Laptop computer — **8471.30**
- Smartphone — **8517.13**
- Drone / unmanned aircraft — **8806**
- Cotton t-shirt — **6109.10**

## Good practice

- Classify by what the good **is**, using its essential character.
- Keep the HS code consistent across the commercial invoice and customs forms.
- When unsure, seek a binding ruling from the destination customs authority rather than
  guessing — and never pick a code purely to lower duty.

# De Minimis & Value Thresholds

ShipSmart knowledge base — advisory guidance; thresholds change, always reconfirm.

## What "de minimis" means

A **de minimis** threshold is the shipment value below which a destination waives duty
(and sometimes tax) on imports. Above it, duty and/or import tax apply, and fuller
documentation is usually required. The value used is typically the declared customs value
of the goods.

## Representative thresholds by destination

- **United States** — US$800 de minimis (Section 321) for most imports per person per day.
- **European Union** — €150 duty de minimis; **VAT applies from €0** (collected via IOSS
  for low-value B2C consignments).
- **United Kingdom** — £135 threshold; above it import VAT and duty handling changes.
- **Canada** — roughly C$20 for tax and C$150 for duty under courier low-value programs.
- **Australia** — A$1,000 import threshold; GST often collected at point of sale below it.

## Why the threshold matters for a shipment

- Below de minimis: usually faster clearance, no duty, lighter paperwork.
- Above de minimis: expect duty/tax, a commercial invoice, and an accurate HS code.
- Splitting one order into multiple parcels to stay under de minimis can be treated as
  **undervaluation/evasion** and penalized.

## High-value handling

For high-value international shipments, a commercial invoice is expected and export
filing may apply (e.g. the US EEI/AES filing above US$2,500 per Schedule B line). Insure
appropriately and keep valuation consistent across all documents.

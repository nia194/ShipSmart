# Legacy

Historical material preserved for reference — none of it is deployed or maintained.

- **`pre-split/`** — the original ShipSmart monorepo (Nx + pnpm workspace: `apps/web`, `apps/api-python`, `infra/`, `supabase/`), as it existed before the system was split into six dedicated repositories. The evolved, maintained versions of everything here live in the component repos mirrored at the repository root.
- **`deployment-docs/`** — deployment-day runbooks, credential guides, and verification scripts from the first production rollout of the pre-split stack. Superseded by each component repo's own README and deploy configuration.
